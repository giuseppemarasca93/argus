# Vision

## Perché esiste Argus

I founder tecnici sanno costruire software, ma spesso scelgono il problema sulla base di impressioni, mode o ricerche frammentarie.

Argus deve trasformare informazioni pubbliche sparse in conoscenza cumulativa utile a prendere decisioni imprenditoriali.

## Cosa deve fare

- osservare fonti selezionate;
- conservare le evidenze originali;
- riconoscere entità e relazioni;
- individuare segnali ricorrenti;
- formulare ipotesi verificabili;
- mostrare sempre da quali fonti deriva una conclusione.

## Cosa non deve fare

- raccogliere indiscriminatamente tutto il web;
- confondere una sintesi AI con un fatto;
- produrre idee generiche senza clienti e problemi identificabili;
- diventare un prodotto SaaS prima di aver risolto il nostro problema interno;
- sostituire interviste e validazione umana.
