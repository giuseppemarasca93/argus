# Roadmap rapida

## Iterazione 0

- definire fonti e domande;
- scegliere stack;
- creare modello dati minimo;
- raccogliere manualmente 20 esempi utili.

## Iterazione 1

- collector RSS e pagine statiche;
- archivio SQLite/PostgreSQL;
- deduplicazione;
- primo report Markdown.

## Iterazione 2

- classificazione tramite LLM;
- estrazione entità;
- sistema di scoring;
- revisione umana degli insight.

## Iterazione 3

- scheduler sul mini PC;
- notifiche e report giornalieri;
- ampliamento controllato delle fonti;
- interfaccia essenziale di consultazione.
